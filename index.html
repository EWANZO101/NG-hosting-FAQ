Minecraft Servers

‚ö†Ô∏è Important First Step - Accept EULA

Before your Minecraft server will start, you MUST accept Mojang's End User License Agreement:

  1. Go to your server's File Manager
  2. Find and edit the file called eula.txt
  3. Change eula=false to eula=true
  4. Save the file and start your server

Basic Configuration

Edit server.properties to customize:

  • ‚Ä¢ motd: Server description
  • ‚Ä¢ difficulty: peaceful, easy, normal, hard
  • ‚Ä¢ gamemode: survival, creative, adventure
  • ‚Ä¢ max-players: Player limit
  • ‚Ä¢ pvp: true/false for PvP
  • ‚Ä¢ online-mode: true for premium accounts only

Connecting to Your Server

Players connect using:

  • ‚Ä¢ Server IP: Found in your panel
  • ‚Ä¢ Port: Usually 25565 (default)
  • ‚Ä¢ Version: Must match server version

Example: 123.456.789.0:25565

üì¶ Installing Plugins (Spigot/Paper)

  1. Download plugin .jar file from trusted sources
  2. Go to File Manager in your panel
  3. Navigate to the /plugins/ folder
  4. Upload the .jar file by dragging it into the browser
  5. Restart your server
  6. Configure plugin settings in /plugins/[plugin-name]/

Trusted Sources: SpigotMC.org, dev.bukkit.org, GitHub

üîß Installing Mods (Forge/Fabric)

  1. Ensure your server is running Forge or Fabric
  2. Download compatible mod .jar files
  3. Go to File Manager in your panel
  4. Navigate to the /mods/ folder
  5. Upload the mod files
  6. Restart your server

Popular Sources: CurseForge.com, Modrinth.com
